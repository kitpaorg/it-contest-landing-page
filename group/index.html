제2회 청소년 IT경시대회 단체접수 안내

  1. 단체 접수 링크로 접속합니다.
    https://kitpa.org/product/it-contest-2nd-group/
  2. 양식을 올바르게 채워줍니다.
    단체명과 담당자명은 이메일로 보내주신 엑셀에 적힌 내용을 그대로 적어주세요.
    인증코드에는 C456를 작성해주세요.
    지역은 단체가 속한 지역을 골라주세요.
    수량은 접수를 진행할 학생의 수를 입력해주세요.

    작성이 완료되었다면 장바구니 버튼을 눌러주세요.

  3. 장바구니 창으로 자동으로 이동되면, 쿠폰 입력 창이 있습니다.
    해당 입력 창에 다시 C456를 입력하신 뒤 쿠폰 적용하기 버튼을 눌러주세요.
    결제 금액이 10% 인하 가격으로 변경되었다면 결제를 진행하시면 됩니다.
